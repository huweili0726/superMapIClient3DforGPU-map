SuperMap iClient3D for WebGL/WebGPU产品介绍

SuperMap iClient 3D for WebGL/WebGPU 是一款在服务式 GIS 架构体系中，无任何插件，跨浏览器的客户端产品。 基于WebGL/WebGPU技术，面向 HTML5 的三维应用开发，快速构建内容丰富、响应迅速、体验流畅的三维真空间应用。
从底层同时支持WebGL和WebGPU两种渲染引擎，对外使用一套API接口便于用户根据实际需求快速切换，降低迁移成本。
