SuperMap iClient3D for WebGL/WebGPU 产品包中范例的本地发布和运行说明

下面将以SuperMap iClient3D 11i(2024) R2 for WebGL WebGPU为例进行介绍。

首先打开SuperMap官网http://support.supermap.com.cn/

点击“软件下载”，打开软件下载界面

点选“网络客户端”→“SuperMap iClient3D 11i(2024)”→“SuperMap iClient3D 11i(2024) R2 for WebGL WebGPU”，打开产品下载界面

将鼠标移至“离线安装包”，选择“官网下载”或“百度云盘下载”，将产品包下载至本地。

官网下载最新的iServer产品包，部署、启动

参照上一步骤，在官网打开软件下载界面，

点选“云GIS”→“SuperMap iServer 11i(2024)”→“SuperMap iServer 11i(2024) R2 for Windows(x64)(绿色部署包)”，打开产品下载界面

将鼠标移至“离线安装包”，选择“官网下载”或“百度云盘下载”，将产品包下载至本地。

将产品包解压放到指定路径。本说明将解压后的产品包放到E:\soft\路径下：

打开“bin”文件夹，双击运行startup.bat启动iServer

产品部署

将SuperMap iClient3D for WebGL/WebGPU产品包解压放到E:\soft\supermap-iserver-11.3.0-windows-x64-deploy\webapps路径下。

打开examples/webgl文件夹，可以查看示范程序的html文件。

运行示范程序

打开Chrome浏览器，输入地址：http://localhost:8090/SuperMap iClient3D 11i(2024) R2 for WebGL WebGPU/examples/webgl/examples.html，可以直接打开示范程序的主页面。
