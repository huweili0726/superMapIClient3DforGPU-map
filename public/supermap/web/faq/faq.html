FAQ

SuperMap iClient JavaScript 11i(2023) 是收费的吗？

SuperMap iClient JavaScript 11i(2023) 核心源代码基于 Apache License 2.0 开源许可协议发布，是完全免费的。可在官网及 GitHub 自由下载和分发使用。

开源库是否会集成到 SuperMap iClient JavaScript 11i(2023) 中，还是用户自行下载？

已将最新兼容版本的三方库在 GitHub、OSChina 托管库中提供（对应 lib 文件夹），用户也可以自行下载，产品介绍页面会提供相关链接。

SuperMap iClient JavaScript 11i(2023) 中的 3D-WebGL 产品和 iClient 3D for WebGL 产品是什么关系？

同一款产品。SuperMap iClient 3D for WebGL 是一款在服务式 GIS 架构体系中，无任何插件，跨浏览器的客户端产品。它基于 Cesium 开源框架,面向 HTML 5 的三维应用开发，可以快速构建内容丰富、响应迅速、体验流畅的三维真空间应用。

EChartsLayer 和 MapVLayer 能否进行多图层叠加？

可以。同时在地图控件中添加两个图层后，会分别在不同的 Canvas 中绘制。

Elasticsearch 部分对接了哪些功能？

详细功能说明请参考 API Doc 中的 SuperMap.Elasticsearch 接口说明。

分布式分析的范例不知道如何设置参数？

默认参数即可，详细参数说明请参考 API Doc 中的 xx.supermap.ProcessingService 接口。
